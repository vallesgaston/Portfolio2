MIS PROYECTOS

App Clima

Wheater App es una SPA(single page aplication) sobre el clima en la cual consume una API externa de openwheatermap.org y actualiza los datos en tiempo real, nos permite obtener informacion meteorologica de cada pais y/o ciudad del mundo. Desarrollada con Javascript, React js y CSs vanilla.

App Fixy

App Fixy es una app movil para encontrar y ofrecer servicios de mantenimiento del hogar, el principal objetivo es generar confianza del usuario, sea cliente que recibe un servicio o profesional que lo brinda. Las tecnologias que fueron utilizadas para el desarrollo: Javascript, React Native, Redux, Expo, React(admin), Nodejs, Express, Sequelize, PostgreSql.

App Pokemons

Pokemons App consiste en una SPA que te permite ordenar, filtrar y ver en detalle informacion de todos los Pokemon traidos de una API externa (https://pokeapi.co/) y tambien de una base de datos, permite crear tu propio Pokemon! Las tecnologias utilizadas fueron: Javascript, React, Redux, CSS, Express, PostreSql, Sequelize, Heroku, Vercel.

App Encriptador-Desencriptador

Esta App consiste en un encriptador de texto en el cual podremos ingresar cualquier palabra o frase y automaticamente se encriptara, tambien podemos seleccionar el boton desencriptar y te lo devolvera. Las tecnologias utilizadas fueron Javascript, HTML, CSS y Trello.
